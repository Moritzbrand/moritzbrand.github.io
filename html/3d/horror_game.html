Lightmare

A horror game created as a university project. The goal is to survive a night in a haunted house. The player must stay in the light, otherwise they will die. Ein Horror-Spiel, das im Rahmen eines Hochschulprojekts entstanden ist. Ziel ist es eine Nacht in einem Geisterhaus zu überleben. Der Spieler muss dabei im Licht bleiben ansonsten stirbt er.

Workflow Arbeitschritte

  • Creating a mesh in Blender Das Mesh in Blender erstellen
  • Texturing the model using Substance Painter Das Modell mittels Substance Painter texturieren
